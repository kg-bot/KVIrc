  • How to install this script
    1. Just use double click on cTopic.kvs
  • What can you do with this script
    1. Find a channels topic fast.
    2. Automatically tell your friends topic of any channel that's not secret.
  • What you can't do with this script
    1. You can't get topic of (+s) secret channels if you're not on it.
    2. You can't change topic with this, use default /topic command.

How to use this script
After you've installed script, following first explanation you can now go to KVIrc and type /ctopic or /ctopic #ChannelName, wait few seconds and you'll see it in it's full potential.
